# 测试

1、编译ros2项目

```shell
cd ros2_project
colcon build
```

2、启动MobaXterm

3、打开仿真环境

```shell
# ShellA
cd ros2_project 
source install/setup.bash
ros2 launch panda_moveit_config gazebo.launch.py
```

4、运行GraphExecuter
```shell
# ShellB
cd GraphExecuter/graph_executer
source /workspaces/ros2_project/install/setup.bash
python3 main.py
```


5、打开控制服务
```shell
# ShellC
cd ros2_project 
source install/setup.bash
ros2 launch yolov8_obb yolov8_obb.launch.py
```
